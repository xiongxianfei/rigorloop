# Material Finding Location Examples

These examples illustrate valid `Location` shapes for material formal review findings. They are not active lifecycle state.

## File And Section

```text
Location: specs/formal-review-recording.md, Requirements R24-R28a
```

## File And Line Range

```text
Location: docs/plans/2026-05-12-review-recording-guardrail-and-downstream-status-settlement.md:63
```

## Artifact And Milestone

```text
Location: active plan M2, Formal Review Skill Recording Output Guardrail
```

## Missing Expected Artifact

```text
Location: missing docs/changes/<change-id>/reviews/plan-review-r1.md
```

## Absence On A Review Surface

```text
Location: plan-review final output, not present: Recording status field
```

The location must be specific enough that a future reader can find the affected surface without chat history.
